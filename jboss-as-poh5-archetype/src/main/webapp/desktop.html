Welcome to JBoss AS 7!

You have successfully deployed a Java EE 6 web application on JBoss AS 7.

This is built using a technique called Plain Old HTML5 (POH5).

  • Pure HTML client
  • JAX-RS GET & POST endpoints for all server interactions
  • HTML5 elements used for page structure
  • HTML5 input type, form validation, and other features
  • CSS3 selectors used for styling
  • Validation & error handling of JAX-RS calls
  • QUnit test suite to validate JavaScript
  • jQuery used to assist
